# LicentaProject
Proiectul meu de licenta - Platforma pentru servicii medicale WebMed
Descriere:
1) Titlu: WebMed (platforma web dedicata serviciilor medicale) 
    Implementare: JS, HTML și CSS, pentru interactivitate și design, MySQL pentru baza de date și Node.js pentru logica serverului. 
2) Contribuția personală: 
    Clase de utilizatori: pacienți, cadre medicale, dispeceri și administratorii platformei. 
    - Pentru pacienți: efectuarea de programari online (atat pentru utilizatorii autentificați, cat și pentru cei care accesează serviciul de oaspeți - fără cont). 
    Lista de medici (gestionata și pregatita de catre dispeceri): se poate verifica disponibilitatea acestora într-un calendar integrat. 
    Istoricul analizelor medicale, utilizând un cod unic generat. 
    Card digital destinat atat fidelizarii, cat și abonamentelor personalizate (istoricul vizitelor, sumele plătite și beneficii precum reduceri sau pachete de servicii medicale). 
    - Pentru medici: Accesul la informațiile pacienților: istoricul de analize medicale, respectand reglementarile GDPR (gazduit local).

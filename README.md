Proiectul meu de licență - Platforma pentru servicii medicale WebMed 🏥
Descriere
Titlu: WebMed (platformă web dedicată serviciilor medicale) 🌐

Implementare:

Tehnologii: JavaScript, HTML și CSS pentru interactivitate și design 💻
Bază de date: MySQL 🗄️
Logica serverului: Node.js 🖥️
Contribuția personală
Clase de utilizatori:
Pacienți 🧑‍⚕️
Cadre medicale 👩‍⚕️
Dispeceri 📞
Administratori 🛠️
Funcționalități pentru pacienți:
Programări online: Disponibile atât pentru utilizatorii autentificați, cât și pentru cei care accesează serviciul de oaspeți (fără cont). 📅
Lista de medici: Gestionată și pregătită de către dispeceri, cu posibilitatea de a verifica disponibilitatea acestora într-un calendar integrat. 🩺
Istoricul analizelor medicale: Accesibil printr-un cod unic generat. 🔍
Card digital: Destinat fidelizării și abonamentelor personalizate, incluzând istoricul vizitelor, sumele plătite și beneficii precum reduceri sau pachete de servicii medicale. 💳
Funcționalități pentru medici:
Acces la informațiile pacienților: Istoricul analizelor medicale, respectând reglementările GDPR (găzduit local). 🔐
